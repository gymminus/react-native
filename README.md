### In terminal from root folder
## Install all packages
```
npm run install-all
```
## Run backend+frontend
```
npm run dev
```
## Run backend
```
npm start
```
## Run frontend
```
cd client
npm start
```

# Setup
### requirements
* npm 14+  
* expo cli  ``` npm i -g expo-cli```
### VS code extensions
* React Native Tools
* React-Native/React/Redux snippets
